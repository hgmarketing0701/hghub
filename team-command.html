📖 Setup — Option B (RECOMMENDED): Apps Script-hosted
  1. You should already have a Google Sheet + Apps Script with the backend code pasted in.
  2. In the Apps Script editor: click + next to Files → HTML → name it exactly Index
  3. Open team-command.html in a text editor (Notepad / VS Code) → Ctrl+A → Ctrl+C
  4. In Apps Script Index file: delete default content → paste → Save (Ctrl+S)
  5. Click Deploy → Manage deployments → pencil edit existing → Version: New version → for max security set "Who has access: Anyone within hggroup.com.my" → Deploy
  6. Copy the Web App URL — share that ONE URL with all 40 staff. No token needed.
  7. Staff open the URL → Google login prompts if not signed in → only @hggroup.com.my accepted → app loads + auto-pulls latest data.

⚠ Photos & signatures stay device-local (Sheets cell size limit). v2.1 will move them to Drive.

  4. Distribute team-command.html file + URL + Token to each staff device
