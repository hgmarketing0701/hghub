🎯 What this system is for — one job

Turn a messy WhatsApp lead into one clean client record with a permanent Client ID.
It does not quote, it does not schedule. It captures the client once and hands a clean BLUE bundle to Smart Quotation.
Everything else in the platform fails if this record is dirty — so keep it small and strict.

4 · How to build it — your stack

Same pattern as your other tools: Google Sheet as the store, Apps Script web app as the brain, single-file HTML front-end.

🟢 Phase 1 MVP — build now

  • One Sheet tab CRM with the columns above
  • HTML form to add / edit a client (manual entry first)
  • Apps Script mints Client ID + stamps date
  • De-dupe check on phone before insert
  • Searchable client list + status filter

🟡 Phase 2 connect

  • WATI webhook → doPost auto-creates leads
  • "Create quote" button → opens Smart Quotation pre-filled by Client ID
  • Status auto-updates from downstream systems
  • Source-channel report (marketing ROI)
